Proyecto de **Machine Learning** desarrollado para la empresa **Telecom X**, enfocado en predecir qué clientes tienen mayor probabilidad de cancelar sus servicios (Churn).  

Este desafío forma parte del programa **Oracle Next Education** en colaboración con **Alura Latam**.

---

# 📌 Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar **modelos predictivos de Machine Learning** capaces de identificar clientes con alta probabilidad de cancelar sus servicios en Telecom X.

A través del **preprocesamiento de datos, análisis de correlación, entrenamiento de modelos y evaluación de métricas**, se construye un pipeline inicial de modelado que permite anticipar la evasión de clientes y apoyar la toma de decisiones estratégicas.

---

# ❓ Problema que resuelve

Telecom X enfrenta una **alta tasa de cancelación de clientes**, lo cual impacta negativamente en sus ingresos y estabilidad del negocio.

Este proyecto busca **predecir qué clientes tienen mayor riesgo de cancelar**, permitiendo a la empresa implementar estrategias de retención antes de que ocurra la cancelación.

# ⚙️ Pipeline de Machine Learning

El proyecto sigue un flujo de trabajo típico de Machine Learning:

### 1️⃣ Preparación de Datos
- Limpieza de datos  
- Tratamiento de valores nulos  
- Codificación de variables categóricas  
- Normalización de datos  

### 2️⃣ Análisis Exploratorio
- Análisis de correlación  
- Identificación de variables relevantes  

### 3️⃣ Modelado Predictivo
Entrenamiento de modelos de clasificación, como por ejemplo:

- Regresión Logística  
- Random Forest  
- Árboles de Decisión  

### 4️⃣ Evaluación de Modelos
Los modelos son evaluados mediante métricas como:

- Accuracy  
- Precision  
- Recall  
- F1 Score  
- Matriz de confusión  

### 5️⃣ Interpretación de Resultados
Se analiza la **importancia de variables** para identificar los factores que más influyen en la cancelación de clientes.
